XOR ALGORITHM

XOR Encryption is an encryption method used to encrypt data and is hard to crack by the brute-force method, i.e generating random encryption keys to match with the correct one. The concept of XOR is to first define the XOR â€“ encryption key and then to perform the XOR operation of the characters in the String with this key which you want to encrypt. To decrypt the encrypted characters we have to perform XOR operation again with the defined key. Here we are encrypting the entire String. The primary reason XOR is so useful in cryptography is that it is "perfectly balanced"; for a given plaintext input 0 or 1, the ciphertext result is equally likely to be either 0 or 1 for a truly random key bit.
